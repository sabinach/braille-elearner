# Braille E-Learner

### 6.835 Spring 2020 - Final Project

#### NOTE:
Make sure your conda, python, and python3 aliases in ~/.bash_profile are commented out before you activate venv!

#### To activate venv
```source venv/bin/activate```

#### To deactivate venv
```deactivate```

#### To run script
```cd final_project```            
```python learn.py```          
```python review.py```

#### To calibrate min/max x
```python calibrate.py```

#### To set parameters
```vim params.py```
